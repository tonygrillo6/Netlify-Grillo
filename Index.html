Heres my GitHub tutorial

Heres how to start up a website using GitHub

  • First,open the terminal located in the launchpad

  • Once opened, enter the command 'cd'to allow changes to be made to the repo folder

  • To add new files enter the code git add

  • These changes will not be seen, but they have been made

  • To check, enter the code "git add"

  • Be sure to commit often and early, to make the changes noticable

  • To mark your first commit type the code "git commit -am first commit"

  • Another thing you must do with your changes push them to GitHub

  • The code for this is git push origin main

  • In addition to pushing changes, you can pull them as well

  • This lets you edit your site on multiple computers

  • The code for this is "git pull origin main"
